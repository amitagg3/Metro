MCRM Utility

It is Java Application developed by MCRM SLS team for automating repetitive and time consuming tasks to make the support process easy and more efficient.

This application is built using Spring Boot, java, HTML, CSS, BootStrap and REST. You can use this application to get the required data required while providing the resolution for an incident.It has various modules such as Invoices, customers, Article etc, each linked with their respective tables in their databases .

Suggestions

If you need huge data for your tasks (more than 100 Mb) please use the SQL Developer for faster results .

Please use this app for quick analysis of incidents and provide suggestions to improve if any.
